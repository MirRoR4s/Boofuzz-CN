环境搭建¶

操作系统环境：WSL-ubuntu22.04

  1. 跟随这篇博客直至支持 Markdown 步骤 Read the Docs

  2. 由于read the docs正进行配置文件的更新，所以现在进行文档托管时要额外编写一个名为.readthedocs.yaml的配置文件。

我的内容如下:

version: 2

build:
  os: "ubuntu-22.04"
  tools:
    python: "3.10"
sphinx:
  configuration: ./docs/conf.py
  fail_on_warning: true

formats:
  - pdf
  - epub

python:
  install:
    - requirements: ./requirements.txt

  1. requirements.txt 和 conf.py 的内容如下：

sphinx-rtd-theme

  1. 为方便起见我选择用 markdown 进行文档编写，但是 markdown 插件装好后每次上传都会报错，所以最后笔者选择直接编写 markdown 并将其转为 rst。通过 pandoc 可以完成文件格式的转化！

代码如下：

sudo apt install pandoc
pandoc -f markdown -t rst input.md -o output.rst

详情可参看官方文档。
