Target

this is a test.

前言

Target 是目标描述符容器。

成员变量与成员方法

__init__

封装连接逻辑。

参数：

  • connection（ITargetConnection）- 到目标系统的连接对象

  • monitors（List[Union[IMonitor, pedrpc.Client]]）- 当前 Target 对象的监视器列表

  • monitor_alive（list）- 当某个监视器处于活跃状态时会调用的一个函数列表

  • repeater（repeater.Repeater）- 发送时所用的 Repeater，默认为 None

  • procmon - 用于添加进程监视器的接口（已弃用）

  • procmon_options - 同上

返回值：

  • Traget对象

close

关闭到目标系统的连接。

参数：

  • None

返回值：

  • None

实现思路：

  1. 调用目标对应套接字的 _target_connection.close() 方法
