class boofuzz.Target(connection, monitors=None, monitor_alive=None, max_recv_bytes=10000, repeater=None, procmon=None, procmon_options=None, **kwargs)[source]ÔÉÅ

Bases: object

Target descriptor container.

Takes an ITargetConnection and wraps send/recv with appropriate FuzzDataLogger calls.

Encapsulates pedrpc connection logic.

Contains a logger which is configured by Session.add_target().

Example

tcp_target = Target(SocketConnection(host=‚Äô127.0.0.1‚Äô, port=17971))

Parameters:

  • connection (itarget_connection.ITargetConnection) ‚Äì Connection to system under test.

  • monitors (List[Union[IMonitor, pedrpc.Client]]) ‚Äì List of Monitors for this Target.

  • monitor_alive ‚Äì List of Functions that are called when a Monitor is alive. It is passed the monitor instance that became alive. Use it to e.g. set options on restart.

  • repeater (repeater.Repeater) ‚Äì Repeater to use for sending. Default None.

  • procmon ‚Äì Deprecated interface for adding a process monitor.

  • procmon_options ‚Äì Deprecated interface for adding a process monitor.
