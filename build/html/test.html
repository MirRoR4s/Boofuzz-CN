环境搭建

操作系统环境：WSL-ubuntu22.04

  1. 跟随这篇博客直至支持 Markdown 步骤 Read the Docs

  2. 安装 MyST 支持 Markdown

pip install myst-parser

之后在conf.py下：

extentions = ["myst_parser"]

详情请参看：Getting started with Sphinx — Read the Docs user documentation

  1. 由于read the docs正进行配置文件的更新，所以现在进行文档托管时要额外编写一个名为.readthedocs.yaml的配置文件。

我的内容如下:

version: 2

build:
  os: "ubuntu-22.04"
  tools:
    python: "3.10"
sphinx:
  fail_on_warning: true

formats:
  - pdf
  - epub

python:
  install:
    - requirements: ./requirements.txt

  1. 也可以不安装 markdown 支持插件，而是直接编写 markdown 并将其转为 rst

代码如下：

sudo apt install pandoc
pandoc -f markdown -t rst input.md -o output.rst

详情可参看官方文档。
